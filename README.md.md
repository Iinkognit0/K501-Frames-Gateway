# iInkognit0 · Vault

Dieses Repository ist der **primäre Gedächtnisraum** des Projekts iInkognit0.

Alles, was hier liegt, ist bewusst abgelegt.
Alles, was fehlt, ist entweder unwichtig oder bewusst nicht gespeichert.

Dieses Vault ist **kein Tagebuch**,  
kein Sammelplatz,  
kein Ideenfriedhof.

Es ist ein **ruhiges Archiv**.

---

## Grundprinzip

> **Bedeutung liegt im Text.  
> Struktur liegt in der Ordnung.  
> Zustand liegt in YAML.  
> Verlauf liegt in SQL.**

Das Projektgedächtnis lebt hier –  
nicht im Chat,  
nicht im Modell.

---
